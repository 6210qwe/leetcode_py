# 题目模板说明

## 模板文件位置
`leetcode_solutions/templates/problem_template.py`

## 使用步骤

1. **复制模板**
   ```bash
   cp leetcode_solutions/templates/problem_template.py leetcode_solutions/by_id/qXXXX.py
   ```

2. **填写题目信息**
   - 题号、标题、难度、链接

3. **填写问题描述**
   - 详细描述题目要求
   - 提供示例和约束条件

4. **填写实现思路**
   - 核心思想
   - 算法步骤
   - 关键点

5. **填写复杂度分析**
   - 时间复杂度和原因
   - 空间复杂度和原因

6. **实现代码**
   - 在函数中实现最优解法
   - 添加必要的注释

## 标准格式示例

参考 `leetcode_solutions/by_id/q0001.py` 的完整格式。
